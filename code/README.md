# Sanskrit Document RAG System (CPU Only)

## Overview
This project implements a Retrieval-Augmented Generation (RAG) system for answering queries over Sanskrit documents using CPU-only inference.

## Features
- Sanskrit-aware preprocessing and chunking
- Vector-based retrieval using FAISS
- CPU-based multilingual LLM generation
- End-to-end modular RAG architecture
